In order to get your Spotify client_id and client_secret:
  1. Create a Spotify Account.
  2. Go to your Spotify Developer Dashboard
  3. Login
  4. Create a new app
  5. The Client ID will be displayed immediately
  6. Click the "Show Client Secret" link to view the Client Secret

Make sure to copy and paste both into the fields here!
